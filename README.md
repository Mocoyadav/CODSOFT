#  CodSoft Web Development Internship - Level 1 Tasks

This repository contains all the tasks completed as part of the **CodSoft Web Development Internship (Level 1)**. Each project demonstrates fundamental web development skills using **HTML, CSS, and JavaScript**.

---

##  Task List

###  Task 1: Portfolio Website

A personal portfolio website built using HTML and CSS.

####  Features:
- **Header Section**
  - Name or logo
  - Short introduction/tagline

- **About Section**
  - Profile image
  - Short bio describing skills and experience

- **Skills Section**
  - List of technical skills

- **Projects Section**
  - Showcase of projects
  - Includes titles, descriptions, and images

- **Resume Section**
  - Downloadable resume (PDF)

- **Contact Section**
  - Email and phone number

- **Footer**
  - Copyright
  - Additional links

---

###  Task 2: Landing Page

A creative and responsive landing page designed using HTML and CSS.

####  Features:
- Clean and modern UI design
- Structured layout using sections and columns
- Proper alignment, spacing, and color palette
- Header and footer included
- Responsive design principles
- Focus on user-friendly interface

---

###  Task 3: Calculator

A basic calculator built using HTML, CSS, and JavaScript.

####  Features:
- Interactive UI with buttons
- Supports:
  - Addition (+)
  - Subtraction (−)
  - Multiplication (×)
  - Division (÷)
- Display screen for input and results
- Implemented using:
  - Event listeners
  - Conditional statements
  - Basic JavaScript logic
- Grid layout for button alignment
